
# 🔺 Triangle Type Analyzer

## 📌 Overview
This project is a Python-based tool that determines the type of a triangle based on:
- its **side lengths**
- its **angles**

It validates input data and classifies triangles into different categories.

---

## ⚙️ Features

### ✅ Classification by sides:
- Equilateral
- Isosceles
- Scalene
- Right-angled triangle

### ✅ Classification by angles:
- Acute triangle
- Right triangle
- Obtuse triangle
- Equilateral triangle

### ✅ Input validation:
- Checks for positive values
- Ensures triangle inequality
- Verifies angle sum (180°)

---

## 🧠 Technologies Used
- Python (core logic)
- Basic input validation
- Mathematical conditions
